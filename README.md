# Personality-Driven Behaviour in Synthetic Social Networks

This repository contains the code, experiments, and analysis developed for a master's thesis focused on synthetic online social networks generated by LLM-based agents.

## Overview

The project investigates how personality traits influence behaviour and structural positions in simulated social networks. Agents are assigned demographic attributes and Big Five personality traits, and interact within a social media simulation environment.

The research combines:

- social network analysis,
- personality-based clustering,
- behavioural modelling,
- statistical validation,
- and comparison with real-world Reddit data.

The main goal is to evaluate whether personality-driven agents produce realistic and reproducible online social structures and behavioural patterns.

The synthetic data used in this study were generated using the YSocial simulation framework, available at: https://github.com/YSocialTwin/YSocial.

---

## Research Objectives

The thesis focuses on:

- analysing node-level behaviour in synthetic social networks,
- studying relationships between personality traits and network structure,
- constructing and analysing personality-based personas,
- comparing personality personas with behaviour-based personas,
- validating simulation realism using Reddit data,
- evaluating behavioural consistency over time,
- and assessing simulation stability across multiple independent runs.

---

## Personas

Agents are clustered into personas based on personality features.

Example personas include:

- socially active and outgoing agents,
- emotionally stable and reserved agents,
- emotionally expressive and friendly agents,
- selective and emotionally reactive agents.

Behavioural personas are additionally created using:
- network metrics,
- activity patterns,
- sentiment and emotional features extracted from generated content.

---

## Methods

The analysis includes:

### Network Analysis
- degree metrics
- betweenness centrality
- eigenvector centrality
- PageRank
- k-core decomposition

### Statistical Validation
- permutation testing
- null distribution analysis
- rank stability analysis

### Comparative Analysis
- synthetic vs real-world Reddit users
- cross-simulation reproducibility
- temporal consistency of agent behaviour
